Expand Object Properties - Based on template strings

Pt.1 - Notation of how to be shorten properties or function

"Expanded object property" is the new function of the way to omit property or function when define it.

1. This is how to define properties by using notation for omitting function.

input

                
                const fuelProp = 100;
                const modelProp = "X-1976";
                
                //if property is the same name, you can omit like below
                const carProp = {
                    fuelProp,
                    modelProp
                };
                /*
                above equals to below
                const car = {
                    fuelProp : fuelProp
                    modelProp : modelProp
                }
                */
                console.log(carProp.fuelProp);//100
                console.log(carProp.modelProp);//X-1976

output

                
                100
                X-1976

2. This is how defining function run by using notation for omitting function.

input

                
                const fuelFunction = 100;
                const modelFunction = "X-1976";
                
                const carFunction = {
                    fuelFunction,
                    modelFunction,
                    run() { // equals to run: function(){}
                        console.log("run");
                    }
                };
                /*
                above equals to below
                const carFunction = {
                    fuelFunction,
                    modelFunction,
                    run: function(){
                        console.log("run");
                    }
                };
                */
                console.log(carFunction.fuelFunction);//100
                console.log(carFunction.modelFunction);//X-1976
                carFunction.run();//run

output

                
                100
                X-1976
                run

if function has any parameter, apply parameter like below:

                
                const car = {
                    fuel,
                    model,
                    run(name){
                        console.log(`run${name}`);
                    }
                }

Pt.2 - How to set dynamic property name

"Expanded object property" can set dynamic property name by using [] in decraration of property.

Try setting dynamic property like the property, named customPropertyName.

input

                
                const fuelDynamic = 100;
                const modelDynamic = "X-1976";
                const customPropertyName = "shippedBy";
                
                const carDynamic = {
                    fuelDynamic,
                    modelDynamic,
                    run(){
                        console.log("run")
                    },
                    [customPropertyName]: "1977"
                    /*
                    Above equals to this below
                        const carDynamic = {
                            carDynamic [customPropertyName]: "1977"
                        }
                    */
                };
                console.log(carDynamic.fuelDynamic);//100
                console.log(carDynamic.modelDynamic);//X-1976
                console.log(carDynamic.shippedBy);//1977
                carDynamic.run();//run

output

                
                100
                X-1976
                1977
                run
  • Before was required defining dynamic properties by using [] after once being store object to value.
    Now defining dynamic properties can be possible to declare all at once by using template strings.
  • You can define dynamic properties more flexibly by using template strings and placeholder at the same time. 
                            
                        const customPropertyName = "shippedBy";
                        const car = {
                            [`${customPropertyName}_dev`]: "1977" //property name is "shippedBy_dev"
                        };
