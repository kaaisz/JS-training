How to Pass Value in ES6

Pt.1 - Default parameter

Like as same with when you do destructuring, set default variable which you will assign, after = to apply default variable.

*Default value is applied only in case of the value was undefined, same as destructuring.

take a data named "fuel" from "car", then assign it to fuel variable.

const { fuel } = car;

same with below

const fuel = car.fuel;

input

            
                function greeting(name = "everyone"){
                console.log("Hello, " + name);
                }
                greeting("Tokyo");//Hello, Tokyo
                greeting("Vancouver");//Hello, Vancouver
                greeting();//Hello, Everyone

output

            
                Hello, Tokyo
                Hello, Vancouver
                Hello, Everyone

Pt.2 - Rest parameter

To being able to handle to variable length values, add ... at the top of parameter.
*Rest parameter can apply to the last parameter.

Make parameter y to being able to handle to variable length values.

input

            
            function add(x, ...y){
            let result = x;
            //add number as many as y which given
            y.forEach(number => {
            result = result + number;
            });
            console.log(result);
            }

output

            
                1
                6
                21
                10

This is the reason why we need to use Rest parameter instead of arguments.

  • Rest parameter includes value which is left. arguments includes all parameters.
  • Because Rest paramater is an array, be able to use method directly like map and sort.
    arguments is not an array.

Understanding Spread Operator

By using Spread operator, array will expand in the place where the operator placed.
To use this operator, put ... at first in an array.

Structure of Spread operator is the same with the structure below.

                
                //example of Spread operator
                array1.push(...array2);
                
                //same with this structure
                array2.forEach((i) => {
                    array1.push(i);
                });

This is a kind of strong operator for making code shorten in case of invoking array, declare array and destructuring.
Spread operator doesn't waste our time at all!
This is the way of how to use Spread operator.

                
                const array = [3, 4, 5];

                //invoke array
                array2.push(...array);

                //declare array
                const newArray = [1, 2 ...array];//[1, 2, 3, 4, 5]

                //destructuring
                [x, ...y] = array;
                console.log(x);//3
                console.log(y);//4, 5

input

            
            const array1 = [1, 2, 3, 4];
            const array2 = [10, 11, 12, 13, 14];
            
            //if without spread operator, you have to adjust array
            // array1.push(array2);//[1, 2, 3, 4, [10, 11, 12, 13, 14]];
            
            ///apply Spread operator
            array1.push(...array2);
            console.log(JSON.stringify(array1));//[1, 2, 3, 4, 10, 11, 12, 13, 14]

output

            
            [1, 2, 3, 4, 10, 11, 12, 13, 14]
