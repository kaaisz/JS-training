Template strings

Pt.1 - How to show template literals

Template literals defines by using backquote(` `)

*To escape backquote, use backslash(\) just before.

input

            
            const message = `Hello`;
            const messageEsc = `Hello\`World`;
            console.log(message);
            console.log(messageEsc);

output

            
            Hello
            Hello`World

Even in multi lines, it's so easy to define by using only backquote.
previous was like this code below.

            const multiLines = Hello, From\nVancity to\nAll over the world.

input

            
            const multiLines = `Hello, From
            Vancity to
            All over the world.`;
            console.log(multilines);

output

            
            Hello, From
            Vancity to
            All over the world.
*this can be include tab or space when you want to see result on console.

Pt.2 - Use placeholder in template strings

In template stirngs, you can use ${}as a placeholder.
${} works only inside of backquotes.

*placeholder means for the things temporarily keep the place to set proper data later.
placeholder can not only refer variable, but also use JavaScript Syntax includes executing function. 
            
            //formula
            `1 + 1 is ${1 + 1}`
            //execute function
            `Hello ${getFullName(name)}`

input

            
            const name = "Everybody";
            const messagePlaceholder = `Hello, ${name}`
            console.log(messagePlaceholder);//Hello everybody
            console.log(`1 + 1 is ${1 + 1}`);

output

            
            Hello everybody
            1 + 1 is 2
