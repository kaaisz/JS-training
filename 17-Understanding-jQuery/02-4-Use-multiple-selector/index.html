Pt.1 - Specify descendant element

Combine each selector by using space to specify descendant element.

$selected = $(".parent .child");

See the Pen qMebWz by suzydp (@suzydp) on CodePen.

Pt.2 - Specify child element

Combine each selector by using > to specify child element.

$selected = $(".parent > .child");

See the Pen <Understanding jQuery&rt;Specify child selector by suzydp (@suzydp) on CodePen.

Pt.3 - Specify adjacent element

Combine each selector by using + to specify adjacent element.

$selected = $(".parent + .child");

See the Pen <Understanding jQuery&rt;Specify adjacent selector by suzydp (@suzydp) on CodePen.

Pt.4 - Specify sibiling element

Combine each selector by using ~ to specify sibiling element.

$selected = $(".parent ~ .child");

Pt.7 - Specify multiple selector

Use , to combine for multiple selector.
Result returns all of the elements which has matched with any selector.

In this case, specify element which has states below:

  • .parent element
  • .child descendant element of .child element.
