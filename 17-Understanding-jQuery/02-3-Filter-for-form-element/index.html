Understanding jQuery

Filters for form element

Pt.1 - Get all of form element

Use :input to get all of form element

*You can target by using selector to get specific element. :input specifies ALL of form control.
*:input targets any elements (even hidden element)

See the Pen <understanding jQuery&rt; Get all form element by suzydp (@suzydp) on CodePen.

Pt.2 - Get specified form element

Use :text, :checkbox, :radio, :button to get specified form element

*:text targets only < input type="text" >
*Need to use Attribute selector if you'll use element added from HTML5
