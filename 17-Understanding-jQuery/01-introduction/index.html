Understanding jQuery

1 - Introduction

Pt.1 - How to declare / write jQuery at start

There are mulitple choice to how to declare(invoke) jQuery at start.
You also have to add CDN or JS file for jQuery to activate.

Pattern 1

            
                jQuery(document).ready(function(){
                    //write code here
                });

Pattern 2

            
                $(function(){
                //write code here
                });

Pattern 3

            
                (function($){
                    //write code here
                });

Pt.2 - Understainding difference between Javascript and jQuery

The code for jQuery is shorter than Vanilla.js

Vanilla.js(JavaScript)

            
                var hello = document.getElementById("hello");
                hello.style.backgroundColor="#9f9";

jQuery

            
                var hello = $("#hello");
                hello.css("backgroundColor", "#9f9");

result is the same like this results below:

Hello!

Pt.5 - Understanding how to choose child class of tag name by using jQuery

So, now as we did, you can also choose child class name by using jQuery.

            
                var elem3 = $(".content__wrap li:first");//equals li:first-child
                elem3.css("backgroundColor", "#f0f");
                
                var elem4 = $(".content__wrap li:eq(1)");//equals li:nth-child(2) *numbers of eq starts from "0(0, 1, 2)".
                elem4.css("backgroundColor", "#e8e");
                
                var elem5 = $(".content__wrap li:last");//equals li:last-child
                elem5.css("backgroundColor", "#b5d");
